Cómo funciona

Un flujo simple para reducir carga: defines la sesión, KOEDUKO la estructura, y tú aplicas y registras.

1
Define
Curso, duración, tipo, área y objetivo general.
2
Genera
Secuencia de pasos y materiales listos para aula.
3
Evalúa
Registro simple por alumno/a con evidencias.
4
Guarda
Historial para reutilizar, ajustar y compartir.

Qué incluye KOEDUKO

Lo esencial para preparar sesiones de Primaria y documentar tu práctica con orden.

Actividades completas
  • Estructura clara: inicio, desarrollo, cierre.
  • Materiales y organización del aula.
  • Variantes y adaptaciones para diferentes ritmos.
Evaluación práctica
  • Criterios e indicadores observables.
  • Evidencias concretas para registrar.
  • Notas rápidas y observaciones.
Fichas imprimibles
  • Actividades para refuerzo y consolidación.
  • Formato preparado para imprimir.
  • Guía breve para el docente.
Historial
  • Accede a tus actividades pasadas.
  • Duplica y adapta en segundos.
  • Construye tu banco de recursos.

Preguntas frecuentes

¿Es una plataforma web?
KOEDUKO es una app. Además, habrá un acceso de escritorio para usuarios registrados, pero por ahora está inactivo.
¿Sirve para cualquier área?
Está pensado para el trabajo habitual de Primaria y para adaptar la planificación a distintas áreas.
¿Puedo imprimir materiales?
Sí, la idea es contar con fichas y materiales listos para imprimir y usar en clase.
